
To run the program, the required Java packages must be installed on the computer to be used. 
Coders for Causes (CFC) has provided all the necessary packages required to run the program in this folder. 

Based on the operating system, the user must install the Java packages present in the one of the sub-folders of Install Java(named according to the OS the user currently uses) in this folder.

For Example, 
if the user uses a Macintosh, they must install the java packages using the jre-8u111-macosx-x64.dmg file located in the sub-folder OSX in Install Java.

Only after complete installation of the required packages can the user use the program.



Note: The Help file must be in the same folder as the Ignite Mentoring program for the "Help' button in the program to work.
